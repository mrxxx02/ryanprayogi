Apa itu GIT?

GIT adalah salah satu version control system yang diciptakan oleh Linus Torvalds GIT mencatat setiap perubahan file di code atau sourcode kita

Apa saja manfaat GIT?

1) Bisa menyimpan seluruh versi source code

2) Berkolaborasi membangun sistem jadi mudah

3) Bisa berkontribusi di proyek open source

4) Memudahkan dalam tracking perubahan

5) Memahami cara deploy modern
