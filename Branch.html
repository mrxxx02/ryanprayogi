Branch

Branch adalah cabang atau copyan dari repo

Manfaat Branch

1) Parallel Development

2) Bisa digunakan untuk memanajemen pengembangan. Misal test branch, unstable branch, stable branch

Pembahasan Branch

1) Git branch: melihat semua branch (local)

2) Git branch --all: melihat semua branch (public)

3) Git branch nama_branch: membuat branch baru

4) Git checkout nama_branch: untuk berpindah branch

5) Git branch -d nama_branch: untuk menghapus branch

Git merge nama_branch

a) Berguna untuk menggabungkan commit dari 2 branch

b) Relatif terhadap branch yang aktif saat ini

c) Untuk melakukan merge harus dipastikan tidak ada file yang belum di commit

Conflict

Yaitu Kondisi yang terjadi ketika merging 2 branch yang memiliki perubahan pada file yang sama. Ketika conflict, kita bisa melakukan commit. Untuk melakukan commit, conflict harus diselesaikan dulu
